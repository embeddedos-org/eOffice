Office Suite AI-Powered 11 Apps

eOffice

Complete office suite with 11 apps: eDocs, eSheets, eSlides, eNotes, eMail, eDB, eDrive, eConnect, eForms, eSway, ePlanner. Powered by eBot AI assistant.

View on GitHub Quick Start

Features

A complete productivity suite for every workflow

eDocs — Word Processing

Rich text editing with formatting, templates, styles, and real-time collaboration support.

eSheets — Spreadsheets

Powerful spreadsheet engine with formulas, charts, pivot tables, and data analysis tools.

eSlides — Presentations

Create stunning presentations with themes, animations, transitions, and speaker notes.

eNotes — Notebooks

Flexible notebook app for note-taking, journaling, and knowledge management.

eMail — Email Client

Full-featured email client with smart inbox, threading, and integrated calendar.

eDrive — Cloud Storage

Secure cloud storage with file sync, sharing, version history, and team folders.

eConnect — Collaboration

Real-time team collaboration with chat, video calls, and shared workspaces.

eForms + eSway + ePlanner

Form builder, interactive content creator, and project planning tools in one suite.

eBot AI Assistant

AI-powered assistant that helps with writing, data analysis, and productivity across all apps.

Real-Time Collaboration

Work together seamlessly with live cursors, comments, and instant sync across devices.

Quick Start

Get up and running in seconds

npm install
npm run dev
